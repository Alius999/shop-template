# E-Commerce (React + Vite)

Быстрый старт интернет-магазина на React с Vite.

## Скрипты

- `npm run dev` — запуск дев-сервера
- `npm run build` — сборка продакшн билда
- `npm run preview` — предпросмотр сборки

## Начало работы

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите проект:
   ```bash
   npm run dev
   ```
3. Откройте `http://localhost:5173`.

Стартовые файлы: `index.html`, `src/main.jsx`, `src/App.jsx`, `src/App.css`.
